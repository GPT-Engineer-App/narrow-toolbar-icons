# narrow-toolbar-icons

A narrow toolbar with icon buttons: 'window' button - on clicked it loads a "tool window" modal pop up, 'search button' when clicked it executes a search function, 'add' button - on clicked it adds a dom element to page.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/narrow-toolbar-icons.git
cd narrow-toolbar-icons
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
